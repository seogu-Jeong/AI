## 1. 제품 서문 (Executive Summary)

### 1.1. 추상적 연산의 기하학적 실체화
대부분의 딥러닝 입문자는 순전파를 단순한 '수식의 나열'로 이해합니다. 하지만 물리학적/수학적 관점에서 순전파는 입력 벡터가 가중치 행렬이라는 **선형 변환(Linear Transformation)**을 거쳐 다른 차원의 공간으로 매핑되고, 활성화 함수라는 **비선형 왜곡(Non-linear Warping)**을 통과하는 연속적인 과정입니다. 본 제품은 2-3-1 구조의 최소 신경망을 통해 이 변환의 전 과정을 시각적으로 증명합니다.

### 1.2. 핵심 가치 (Core Values)
*   **Dimensional Fluidity:** $\mathbb{R}^2 \to \mathbb{R}^3 \to \mathbb{R}^1$로 이어지는 차원 축소와 확장의 물리적 의미 시각화.
*   **Real-time Topology Interaction:** 네트워크 구조(Topology)의 변화가 행렬 원소값에 미치는 영향을 즉각적으로 동기화.
*   **Pedagogical Transparency:** '블랙박스'로 치부되는 신경망 내부의 연산 과정을 HUD(Head-Up Display) 수식으로 투명하게 공개.

---

## 2. 타겟 유저 분석: 물리학과 학부생

### 2.1. 유저의 수학적 배경
*   행렬 내적($\mathbf{W}\mathbf{x}$), 기저 변환(Basis Transformation), 고유값 문제 등에 익숙함.
*   텐서 연산의 차원 일치(Dimension Matching) 조건에 민감함.

### 2.2. 요구사항
*   "단순히 결과값만 보여주는 것이 아니라, 행렬의 각 행(Row)이 입력 벡터와 어떻게 내적되어 각 뉴런의 활성화 값으로 치환되는지 단계별로 보고 싶다."

---

## 3. 심층 유저 시나리오: 연쇄적 상태 변화 (Chain Reaction)

### 3.1. 단계 1: 입력 벡터의 섭동 (Perturbation)
1.  사용자가 좌측 패널의 입력 노드 $x_1, x_2$ 슬라이더를 조절합니다.
2.  **시각적 반응:** 입력 벡터 $\mathbf{x} = [x_1, x_2]^T$의 크기가 노드의 밝기로 표시되며, 행렬 수식 HUD에서 $X$ 행렬의 수치가 실시간으로 변합니다.

### 3.2. 단계 2: 은닉층으로의 선형 사상 (Mapping to Hidden Layer)
1.  입력값이 변함에 따라 은닉층(3개 노드)으로 연결된 6개의 엣지(Edge)를 통해 '에너지'가 흘러가는 애니메이션이 발생합니다.
2.  **증명 과정:** 사용자는 중앙의 **Bar Chart**를 통해 은닉층 노드 $h_1, h_2, h_3$의 활성화 값이 변하는 것을 관찰합니다. 
3.  **수학적 확인:** HUD에 표시된 가중치 행렬 $\mathbf{W}^{(1)} \in \mathbb{R}^{3 \times 2}$의 각 행 벡터와 입력 벡터 $\mathbf{x}$의 내적 합산 과정을 눈으로 추적합니다.

### 3.3. 단계 3: 출력층으로의 수렴 (Convergence to Scalar)
1.  은닉층의 3개 상태 벡터가 다시 가중치 $\mathbf{w}^{(2)} \in \mathbb{R}^{1 \times 3}$와 결합하여 최종 스칼라 값 $y$를 산출합니다.
2.  사용자는 "3차원 공간의 정보가 어떻게 다시 1차원 직선상의 점으로 투영(Projection)되는가?"를 기하학적으로 이해합니다.

## 4. 상세 기능 명세 (Functional Specifications)

### 4.1. 인터랙티브 네트워크 토폴로지 다이어그램 (3)
*   **노드 시각화:** 활성화 값($a_i$)이 클수록 노드의 테두리 광원(Glow)이 강해짐.
*   **엣지 시각화:** 가중치($w_{ij}$)의 절대값이 클수록 선의 굵기가 굵어지며, 양수는 실선, 음수는 점선으로 표현.
*   **상호작용:** 노드를 클릭하면 해당 노드에 할당된 바이어스($b$) 값을 즉석에서 수정할 수 있는 미니 팝업 노출.

### 4.2. 행렬 수식 HUD (Head-Up Display) (3)
*   **위치:** 네트워크 다이어그램 상단에 오버레이.
*   **내용:** $\mathbf{a}^{(1)} = \sigma(\mathbf{W}^{(1)}\mathbf{x} + \mathbf{b}^{(1)})$ 형식의 수식을 실제 수치와 함께 LaTeX로 렌더링.
*   **동기화:** 슬라이더 조작 시 0.01초 이내에 수식 내 숫자가 업데이트됨.

### 4.3. 실시간 바 차트 모니터 (2)
*   **은닉층 전용:** 3개 노드의 활성화 상태를 $0 \sim 1$ 범위의 막대 그래프로 상시 표시.
*   **대조군:** 활성화 함수 적용 전($z$)과 후($a$)를 나란히 배치하여 비선형 왜곡의 효과를 시각화.

*   모든 연산은 `numpy.float64`를 사용하며, UI에는 소수점 3자리까지 반올림하여 표시.

## 7. 교육적 성공 지표

*   입력 노드 하나를 변경했을 때 전파되는 파동(Ripple Effect)을 통해 연쇄 법칙의 물리적 기초를 이해함.
