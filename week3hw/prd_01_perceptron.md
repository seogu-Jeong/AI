## 1. 제품 서문 (Executive Summary)

### 1.1. 교육적 패러다임의 전환
본 프로젝트는 인공지능의 가장 기초적인 단위인 '퍼셉트론'을 수학적 추상화의 영역에서 끌어내어, 물리학의 '전기장과 전위'라는 물리적 실체로 재정의합니다. 물리학과 학생들에게 가중치 업데이트는 단순한 수치 최적화가 아닌, 공간의 이방성(Anisotropy)을 조절하여 시스템의 총 에너지를 최소화하는 '필드 엔지니어링'임을 시각적으로 증명합니다.

### 1.2. 핵심 가치 제안 (Unique Value Proposition)
*   **물리적 상동성(Physical Homology):** 신경망의 로짓($z$)을 전위($V$)로, 가중치($w$)를 전기장($E$)의 강도로 치환하여 해석.
*   **위상적 직관(Topological Intuition):** 선형 분리의 한계를 위상적 결함(Topological Defect) 관점에서 시각화.
*   **연구급 정밀도:** 실시간 60FPS 상호작용과 소수점 8자리의 수치적 정밀도 동시 확보.

## 2. 상세 유저 페르소나 (Target Persona)

### 2.1. 물리학과 학부생 'A군'
*   **배경:** 전자기학 I, II를 수강하여 맥스웰 방정식과 전위 경계값 문제에 익숙함.
*   **갈증:** "왜 AI는 데이터를 분류할 때 직선을 사용하는가? 그 직선 뒤에 숨겨진 공간의 물리적 성질은 무엇인가?"에 대한 해답을 찾고 싶어 함.
*   **목표:** 퍼셉트론의 결정 경계가 물리적 전위의 절점선(Nodal Line)임을 이해하고, XOR 문제의 분리 불가능성을 공간의 대칭성 깨짐 관점에서 설명함.

## 3. 심층 유저 시나리오 (Deep User Scenarios)

### 3.1. 시나리오 1: 사인 곡선 경계의 위상적 한계 돌파
1.  **데이터 생성:** 사용자가 하단 입력창에 `y = 2 * sin(x)`를 입력합니다.
2.  **자동 샘플링:** 시스템은 해당 곡선을 경계로 상단 50개(Red), 하단 50개(Blue)의 점을 가우시안 노이즈를 섞어 생성합니다.
3.  **물리 모드 활성화:** 사용자가 'Physics Mode'를 켭니다. 이제 슬라이더는 $w$ 대신 유전율($\epsilon$)과 전하량($q$) 단위로 표시됩니다.
4.  **장 형성:** 가중치 슬라이더를 조작하며 전위 히트맵을 관찰합니다. 직선인 결정 경계가 사인 곡선의 굴곡을 따라가지 못해 발생하는 '오분류 영역'을 전기장의 세기 분포(V/m)로 확인합니다.
5.  **통찰:** "단일 전기장은 공간을 한 방향으로만 기울일 수 있기에, 사인파처럼 굽이치는 데이터는 분리할 수 없다"는 물리적 결론에 도달합니다.

### 3.2. 시나리오 2: XOR 지형에서의 에너지 최소화 실패
1.  **데이터 로드:** XOR 데이터셋을 선택합니다.
2.  **분석 창 오픈:** 별도 창으로 'Loss Landscape Analyzer'를 띄웁니다.
3.  **지형 관찰:** $w_1, w_2$ 축에 따른 MSE 손실 지형이 '안장점(Saddle Point)'을 포함한 복잡한 형태로 나타납니다.
4.  **최적화 시도:** 사용자가 슬라이더를 움직일 때마다, 지형 위의 빨간 공(현재 상태)이 전역 최소점(Global Minimum)을 찾지 못하고 평탄한 계곡 사이를 영원히 진동하는 현상을 목격합니다.

## 4. 상세 기능 명세 (Detailed Functional Requirements)

### 4.1. 3D Contour Projection 시스템 (1-1)
*   **기능:** 2D 등고선(Contour)을 3D 전위 곡면 위에 실시간 텍스처로 매핑.
*   **상호작용:** 사용자가 2D에서 가중치를 조절하면, 3D 곡면이 실시간으로 틸팅(Tilting)되며 곡면 위의 등고선 무늬가 위상적으로 수축/팽창하는 모습을 시연.
*   **물리적 의미:** 전위 차가 클수록 등고선 간격이 좁아지는 전기장 세기($E = -\nabla V$)의 시각적 증명.

### 4.2. 수학적 함수 기반 데이터 생성기 (1-2)
*   **입력 인터페이스:** `numpy` 문법을 지원하는 수식 입력창.
*   **생성 알고리즘:** 입력된 함수 $f(x)$에 대해 $y > f(x)$ 영역과 $y < f(x)$ 영역을 구분하여 랜덤 포인트 생성.
*   **노이즈 조절:** 표준편차($\sigma$) 슬라이더를 통해 경계 부근의 모호성(Ambiguity) 조절 기능.

### 4.3. Physics-to-AI 컨버터 (1-3)
*   **단위 변환:** 가중치 벡터 $\vec{w}$를 전기장 벡터 $\vec{E}$로, 편향 $b$를 기준 전위 $V_0$로 변환.
*   **수치 출력:** 결정 경계로부터 특정 거리 $d$에서의 전위 $V$와 전기장 세기 $E$를 물리 단위($V, V/m$)로 화면 우측 상단에 실시간 렌더링.

### 4.4. 샘플링 기반 손실 지형 분석 (1-4)
*   **그리드 샘플링:** $w_1, w_2$ 평면을 $50 \times 50$ 그리드로 나누어 각 점에서의 Loss를 병렬 연산.
*   **보간 기술:** `Bicubic Interpolation`을 사용하여 $500 \times 500$ 수준의 초고해상도 곡면으로 부드럽게 시각화.
