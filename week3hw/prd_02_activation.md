## 1. 제품 서문 (Executive Summary)

### 1.1. 비선형성의 심연을 시각화하다
딥러닝의 핵심은 선형 변환의 한계를 극복하는 '비선형 활성화 함수'에 있습니다. 본 앱은 미적분학의 연쇄 법칙(Chain Rule)을 기반으로, 기울기(Gradient)가 층을 통과하며 어떻게 감쇄(Vanishing)되거나 증폭(Exploding)되는지를 물리적 에너지 흐름으로 재해석하여 시각화합니다.

### 1.2. 핵심 가치 제안 (Unique Value Proposition)
*   **Dynamic Differentiation:** 커스텀 함수에 대한 실시간 기호 미분 및 도함수 매핑.
*   **Multi-layer Flow Analysis:** 심층 신경망의 기울기 전파 과정을 1:1 대응 그래프로 분석.
*   **Singularity Explorer:** ReLU 등의 불연속점에서 Sub-gradient의 영향을 직접 튜닝.

## 2. 타겟 유저 및 인지적 목표

### 2.1. 타겟: 연쇄 법칙을 정복하려는 물리학/공학도
*   **인지적 목표:** 복합 함수의 미분값이 각 성분 함수의 도함수 곱으로 결정됨을 직관적으로 이해.
*   **실험 목표:** 활성화 함수의 형태가 '기울기 소실' 문제의 근본 원인임을 실험적으로 증명.

---

## 3. 심층 유저 시나리오 (Deep User Scenarios)

### 3.1. 시나리오 1: Sigmoid의 늪(Saturation) 탈출하기
1.  **설정:** 사용자가 `Sigmoid` 함수를 선택하고 `Multi-layer Flow` 모드에서 레이어를 4개로 설정합니다.
2.  **입력:** 글로벌 입력 슬라이더 $x$를 $5.0$으로 이동시킵니다.
3.  **관찰:** 각 층의 출력값이 1.0에 가까워지면서(포화), 각 층의 도함수 값이 $0.01$ 이하로 떨어지는 것을 확인합니다.
4.  **전파:** 마지막 층에서 생성된 기울기가 역방향으로 전달될 때, 각 층을 지날 때마다 $0.01$이 계속 곱해져 초기 층에서는 기울기가 소멸하는 '기울기 소실'을 미분 그래프의 Bar Chart로 목격합니다.
5.  **해결:** 모든 층을 `ReLU`로 교체하여 $x > 0$ 영역에서 기울기가 1.0으로 보존되는 'Deep Flow' 현상을 대조 확인합니다.

### 3.2. 시나리오 2: 커스텀 함수와 Sub-gradient 탐구
1.  **입력:** 사용자가 커스텀 함수 입력창에 `exp(-x^2) * sin(3x)`를 입력합니다.
2.  **분석:** 시스템이 즉석에서 도함수를 계산하여 원형 그래프 옆에 띄웁니다.
3.  **불연속점 실험:** `Step Function`을 선택하고 $x=0$에서의 미분 계수 $\gamma$ 슬라이더를 $0.5$에서 $1.0$으로 조절하며 도함수 그래프의 불연속점 처리가 어떻게 변하는지 관찰합니다.

## 4. 상세 기능 명세 (Detailed Functional Requirements)

### 4.1. Multi-layer Gradient Flow 시뮬레이션 (2-1)
*   **기능:** $n$개(최대 5개)의 활성화 함수를 직렬 연결한 가상의 채널 형성.
*   **시각화:** 각 레이어의 도함수 함숫값을 '기울기 통과율(Transparency)'로 환산하여 선의 굵기나 밝기로 표현.
*   **수치 출력:** 연쇄 법칙의 최종 결과값 $\prod f'_i(x)$를 실시간 계산하여 표시.

### 4.2. 커스텀 함수 기호 미분 엔진 (2-2)
*   **기능:** 사용자가 텍스트로 입력한 수학 식을 `SymPy`로 해석하여 도함수를 도출.
*   **제약:** `x` 변수만 허용하며, 기본적인 삼각함수, 지수함수, 로그함수 지원.

### 4.3. Sub-gradient Tuner (2-3)
*   **기능:** 미분 불가능한 점(ReLU의 0, Step의 0 등)에서 임의의 미분값을 할당하는 기능.
*   **조절 범위:** $[0, 1]$ 범위의 연속 슬라이더 제공.

### 4.4. PyQtGraph 기반 120FPS 렌더링 (2-4)
*   **기능:** 슬라이더 조작 시 원형 함수와 도함수 그래프, 그리고 접선(Tangent Line)이 잔상 없이 부드럽게 움직임.
*   **기술:** OpenGL 가속 및 PyQtGraph의 고속 데이터 업데이트 매커니즘 활용.
