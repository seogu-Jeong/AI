# PNU Physics Webpage (Velvet Edition) - week2-1

이 프로젝트는 부산대학교 물리학과를 소개하는 고품격 다중 페이지 웹 애플리케이션입니다. **FastAPI** 백엔드와 **Tailwind CSS**, **고급 벨벳 레드 테마**를 사용하여 전문가급 UI/UX를 구현했습니다.

## 🍷 핵심 특징 (Tech & Design)

1.  **벨벳 레드 & 골드 테마**: 
    - 딥 레드(#4A0E17)와 럭셔리 골드(#D4AF37) 색상 조합으로 학술적 권위와 고급스러움을 강조.
    - `style.css`에 노이즈 필터와 그라데이션을 적용하여 실제 벨벳 질감을 시각적으로 구현.
2.  **실제 작동하는 다중 라우팅**:
    - `main.py`에 정의된 라우터를 통해 모든 네비게이션 링크(학과소개, 교수진, 연구분야 등)가 실제 페이지로 이동합니다.
3.  **최신 교수진 및 연구 데이터**:
    - 홍덕기, 유인권, 문한섭 교수님 등 실제 부산대 물리학과 교수진의 실명과 세부 전공 정보를 포함.
    - 입자물리, 응집물질, 광학, 통계물리 등 4대 주요 연구 분야를 상세히 기술.
4.  **Bulletproof 시스템**:
    - 코드가 위치한 경로를 자동으로 추적하는 절대 경로 시스템 적용.
    - 실행 시 `fastapi`, `uvicorn` 등 필수 패키지가 없으면 자동으로 설치하는 기능 탑재.

```text

```

## 💎 전문가급 디테일
- **Typography**: modern Korean 폰트인 'Pretendard'를 사용하여 가독성을 극대화했습니다.
- **Micro-interactions**: 카드 호버 시 부드러운 이동 애니메이션과 골드 테두리 강조 효과가 적용되었습니다.
- **Glassmorphism**: 상단 네비게이션 바에 투명 블러 효과를 주어 세련된 느낌을 더했습니다.
