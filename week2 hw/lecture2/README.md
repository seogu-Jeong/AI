# 🎸 MIT 6.S191 Lab 1: AI Music Generation (PyTorch)

## 🚀 프로젝트 소개 (Overview)
이 프로젝트는 MIT 6.S191 딥러닝 부트캠프의 첫 번째 랩(Lab 1) 과제 완성본입니다. 
순환 신경망(RNN, 구체적으로는 LSTM)을 사용하여 수천 곡의 아일랜드 포크송(ABC 표기법)을 학습한 뒤, 세상에 단 하나뿐인 새로운 멜로디를 작곡하고 실제 들을 수 있는 `.wav` 오디오 파일로 추출하는 인공지능 모델을 구현했습니다.

## 💻 권장 실행 환경 (Environment)
이 코드는 오디오 생성 및 변환을 위한 외부 패키지(`abcmidi`, `timidity` 등) 설치가 필요하므로 **Google Colab** 환경에서 실행하는 것을 강력하게 권장합니다. 

* **권장 플랫폼:** Google Colab
* **하드웨어 가속기:** GPU 필수 (Colab 상단 메뉴: `런타임` -> `런타임 유형 변경` -> `하드웨어 가속기: T4 GPU` 등 선택)

## 🛠️ 사용 방법 (How to Run)
1. Google Colab에서 새 노트북(New Notebook)을 엽니다.
2. 제공된 파이썬 스크립트 코드를 빈 셀에 통째로 복사하여 붙여넣습니다.
3. `Shift + Enter`를 누르거나 좌측의 재생 버튼을 눌러 코드를 실행합니다.
4. 아래의 과정이 원스톱으로 자동 진행됩니다:
   - 패키지 설치 및 데이터셋 다운로드
   - 데이터 전처리 및 미니 배치(Mini-batch) 생성
   - LSTM 모델 훈련 (기본 설정 2,000 Iterations)
   - 새로운 ABC 악보 생성 및 오디오(.wav) 변환

## 🎧 결과물 (Output)
코드가 성공적으로 끝까지 실행되면 다음 결과물을 얻을 수 있습니다.
* **오디오 플레이어:** Colab 화면 하단에 재생 버튼이 생성되어 AI가 작곡한 곡을 즉시 들어볼 수 있습니다.
* **WAV 파일 저장:** Colab 좌측 폴더(파일 브라우저) 탭을 확인하면 `my_ai_song_0.wav` 파일이 생성되어 있습니다. 이를 다운로드하여 과제로 제출할 수 있습니다.

## 🧠 핵심 학습 개념 (Key Concepts)
* **Sequence Modeling:** 시간의 흐름(순서)이 존재하는 텍스트/음악 데이터를 다루는 딥러닝 기법.
* **LSTM (Long Short-Term Memory):** 바닐라 RNN의 장기 의존성(Long-term dependency) 한계를 극복하고 과거의 문맥을 유지하며 다음 토큰(음표)을 예측하는 아키텍처.
* **Sampling with Multinomial Distribution:** 모델이 무조건 가장 확률이 높은 음표만 선택하여 반복적인 멜로디를 생성하는 것을 막기 위해, 확률 분포를 기반으로 무작위성을 부여하는 테크닉.
