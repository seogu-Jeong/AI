## Goal

One standalone PySide6 desktop app (`phys0304нҶөн•©/`) with three tabs implementing:
- **MOD-02** вҖ” Projectile Motion Regression: NN learns (vвӮҖ, Оё, t) вҶ’ (x, y)
- **MOD-04** вҖ” Pendulum Period Prediction: NN learns (L, ОёвӮҖ) вҶ’ T with QPainter animation
- **MOD-05** вҖ” Air Resistance Projectile: RK4 simulation + NN learns (vвӮҖ, Оё) вҶ’ range R

---

## Architecture

Same layered pattern as physicsai and funcai:

```
Presentation   PySide6 widgets, MainWindow, module tabs
App Logic      BaseModule FSM (IDLE/TRAINING/TRAINED/DIRTY/ERROR), Template Method
Physics Eng.   rk4.py, projectile.py, pendulum.py вҖ” pure NumPy, no Qt
ML Engine      models.py, data_generators.py, training_worker.py, loading_worker.py
Infrastructure NumPy, Matplotlib (QtAgg backend), QPainter (MOD-04 only)
```

Layers communicate only via Qt Signal/Slot. Physics and ML layers have zero Qt imports.

## Module Specifications

### MOD-02: Projectile Motion Regression

**Physics:** Vacuum trajectory вҖ” x(t)=vвӮҖcos(Оё)t, y(t)=vвӮҖsin(Оё)tвҲ’ВҪgtВІ

**ML model:** Input(3) вҶ’ Dense(128,relu)+DO(0.1) вҶ’ Dense(64,relu)+DO(0.1) вҶ’ Dense(32,relu)+DO(0.1) вҶ’ Dense(2,linear)

**Data:** vвӮҖвҲҲU[10,50]m/s, ОёвҲҲU[20В°,70В°], tвҲҲU[0,0.9В·T_f]; n=2000; Gaussian noise Пғ configurable [0,2]m; y<0 samples filtered

**Plot layout:** GridSpec 2Г—2, height_ratios=[2,1]
- gs[0,:] merged: trajectory comparison (True blue, NN red dashed), xlimвүҘ0, ylimвүҘ0, annotate max-height and max-range
- gs[1,0]: x(t) vs time
- gs[1,1]: y(t) vs time

**Multi-condition overlay:** Up to 5 conditions, tab10 palette, "Add Condition" + "Clear Conditions" buttons

**Post-training:** slider change вҶ’ re-predict (no retrain) within 500ms

**Requirements covered:** FR-MOD02-01..17

---

### MOD-04: Pendulum Period Prediction

**Physics:** dВІОё/dtВІ = -(g/L)sin(Оё); RK4 state [Оё(rad), Пү(rad/s)]; 3 periods; T_exact = T_smallГ—(1+ОёвӮҖВІ/16+11ОёвӮҖвҒҙ/3072)

**ML model:** Input(2) вҶ’ Dense(64,relu)+DO(0.1) вҶ’ Dense(32,relu)+DO(0.1) вҶ’ Dense(16,relu) вҶ’ Dense(1,linear); metrics=['mae','mape']

**Data:** LвҲҲU[0.5,3.0]m, ОёвӮҖвҲҲU[5В°,80В°]; n=2000; multiplicative noise N(1,0.01)

**Layout (right panel):** QSplitter horizontal
- Left 40%: `PendulumCanvas` (QPainter вҖ” pivot, rod, bob, trail 30pts, angle arc+label)
- Right 60%: QSplitter vertical
  - Top: Оё(t) plot (blue line, red dashed period markers, black dashed equilibrium)
  - Bottom: Phase space (Оё vs Пү, green line, red start marker)

**Period panel (in left scroll area below params):**
- T_small (small-angle), T_exact (elliptic), T_pred (NN) вҖ” all three with relative error vs T_exact
- ОёвӮҖ>30В° вҶ’ T_small error highlighted amber
- Horizontal bar chart comparing three values

**Animation controller:** QTimer 33ms (30FPS), Play/Pause button, Space shortcut, speed 0.25Г—/0.5Г—/1Г—/2Г—/4Г—, auto-start on param change

**Isochronism demo:** toggle button вҶ’ second pendulum ОёвӮҖ=5В° in orange, О”T display

**Analysis tab:** MAPE vs ОёвӮҖ curve, True vs Predicted scatter

**Requirements covered:** FR-MOD04-01..36

---

### MOD-05: Air Resistance Projectile

**Physics:** RK4 state [x,y,vx,vy]; derivatives: [vx, vy, -k|v|vx, -g-k|v|vy]; kвҲҲ[0,0.20] default 0.05

**ML model:** Input(2) вҶ’ Dense(64,relu) вҶ’ Dense(64,relu) вҶ’ Dense(32,relu) вҶ’ Dense(1,linear)

**Data generation:** `LoadingWorker` QThread runs 2000 RK4 simulations; progress bar updates every 100 sims; vвӮҖвҲҲU[10,100]m/s, ОёвҲҲU[10В°,80В°]

**Plot layout:** GridSpec 2Г—2
- gs[0,0]: Trajectory (vacuum blue dashed, RK4 air red solid, AI star green вҳ… at R_pred)
- gs[0,1]: Range vs Angle (vacuum theory blue dashed, RK4 red solid, AI green dots, optimal angle red vline)
- gs[1,0]: AI performance scatter (True vs Pred, y=x reference, RВІ in title)
- gs[1,1]: Training loss curve

**Live info panel:** Vacuum Range, Air Resistance Range, AI Predicted Range, Range Reduction (%), Optimal Angle вҖ” updates on vвӮҖ/Оё/k slider change

**k-change warning:** "Drag coefficient changed. Retraining required for accurate AI predictions."

**Requirements covered:** FR-MOD05-01..14
